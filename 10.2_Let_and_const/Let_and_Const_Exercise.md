What is the difference between var and let?

- Var variables can be reassigned and redeclared. Let variables can be reassigned but not redeclared. 
- Var variables get added to the window object, let variables don’t get added to the window object.
- Var variables are function scoped, let variables are block scoped.
- Var variables are hoisted, let variables aren’t hoisted.

What is the difference between var and const?

- Var variables can be reassigned and redeclared. Const variables can’t be reassigned or redeclared. 
- Var variables get added to the window object, const variables don’t get added to the window object.
- Var variables are function scoped, const variables are block scoped.
- Var variables are hoisted, const variables aren’t hoisted.
